HTML & CSS Notes

1. What is HTML?

HTML (Hypertext Markup Language) is the standard language used to create webpages. It structures the content of a webpage by defining elements such as headings, paragraphs, links, images, forms, and more. HTML uses tags to mark up content. Tags usually come in pairs, with an opening tag and a closing tag. Some tags are self-closing (e.g., <img>).

2. Structure of an HTML Document

  • <!DOCTYPE html>: Declares that the document is an HTML5 document.
  • <html>: The root element of the HTML document.
  • <head>: Contains meta-information about the webpage such as character encoding, title, and links to external files (like CSS).
  • <body>: Contains the actual content of the webpage such as text, images, links, and more.

3. HTML Tags

  • <h1> to <h6>: Used to define headings, with <h1> being the largest and most important.
  • <p>: Used to define blocks of text.
  • <a>: Used to define hyperlinks, which allow users to navigate to other pages or websites.
  • <img>: Used to embed images on the page.
  • <ul>, <ol>, <li>: Used to define unordered (bulleted) and ordered (numbered) lists.

4. HTML Attributes

  • id: Uniquely identifies an element on the page (used for JavaScript manipulation or CSS styling).
  • class: Groups elements together for styling.
  • href: Specifies the destination URL for links.
  • src: Specifies the source file for images.
  • alt: Provides alternative text for images for accessibility.

5. Forms and Input Elements

  • <form>: The container for form elements.
  • <input>: Defines different types of input fields like text, password, email, etc.
  • <label>: Associates a label with a form element.
  • <textarea>: Used for multi-line text input.
  • <select>: Creates dropdown menus.

6. Semantic HTML

Semantic HTML uses meaningful tags that describe the content they enclose, improving SEO and accessibility. Examples include: <article>, <section>, <header>, <footer>, and <nav>.

7. Tables in HTML

  • <table>: Defines the table.
  • <tr>: Defines a row in the table.
  • <td>: Defines a cell (data) in the table.
  • <th>: Defines a header cell in the table.

8. HTML Comments

HTML comments are used to add notes to the code and do not display on the webpage. Syntax: <!-- This is a comment -->.

9. Best Practices in HTML

  • Keep the structure semantic for better readability, accessibility, and SEO.
  • Use proper indentation to make your code easy to read.
  • Avoid inline styles and scripts when possible, use external CSS and JavaScript files.

CSS Notes

1. What is CSS?

CSS (Cascading Style Sheets) is used to control the appearance and layout of HTML elements. CSS allows you to style various aspects of a webpage, including colors, fonts, spacing, positioning, and more.

2. Basic CSS Syntax

p {
        color: red;
        font-size: 16px;
    }

3. CSS Selectors

  • Element Selector: Targets all instances of an HTML element (e.g., p for all paragraphs).
  • Class Selector: Targets elements with a specific class. It starts with a dot (.) (e.g., .highlight).
  • ID Selector: Targets an element with a specific ID. It starts with a hash (#) (e.g., #main-header).
  • Universal Selector: Targets all elements on the page (e.g., *).
  • Descendant Selector: Targets elements that are inside other elements (e.g., div p targets all <p> elements inside <div>).

4. The CSS Box Model

The CSS box model describes the rectangular boxes generated for elements, consisting of:

  • Content: The actual content of the element.
  • Padding: The space around the content.
  • Border: The edge around the padding.
  • Margin: The space outside the border.
div {
        padding: 20px;
        border: 1px solid black;
        margin: 30px;
    }

5. Layout Techniques

  • Flexbox: A one-dimensional layout model for aligning and distributing space among items in a container.
  • CSS Grid: A two-dimensional layout system that controls both rows and columns.
  • Positioning: Defines how an element is placed on the page.

6. Styling Text

  • font-family: Specifies the font.
  • font-size: Defines the size of the text.
  • color: Defines the text color.
  • line-height: Sets the height of each line of text.
  • text-align: Aligns the text (e.g., left, right, center).
